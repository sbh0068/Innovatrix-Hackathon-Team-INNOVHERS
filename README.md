TASK TRIAL

It's a platform to bridge opportunity gap of tier 2/3 students.

OVER VIEW
     - A web platform connecting Students and Msmes.
     - Focuses on tier 2/3 college students.
     - Helps skilled students get noticed without any IIT/NIT tag.

AIM
   - Bridge opportunity gap between students and recruiters.
   - Provide equal hiring opportunities based only on skills.

FEATURES

 -Students' Side :-
    - SignUp and Login
    - View Job Postings
    - A Dashboard which includes :
       - Task Completed
       - Applications
     - Navigation Panel :
        - Dashboard
        - My Applications
        - Skill Tasks
        - Profile 
        - Logout
        
-MSMEs' Side :-
   -Post Job Opportunities
   - Add Job Details :
        - Job Requirements
         - Job description
         - Skills Required
          - Salary
    - View Student's Profile

AUTHENTICATION
   - New Users => Register and Enter Details
   - Existing Users => Login and Go to Dashboard

     WORKING FLOW
        - User visits welcome page
        - Selects role (Student/Msme)
        - Choose Login or SignUp
        - Gets redirected to respective dashboard
        - Uses platform features

PROJECT STRUCTURE
  - /frontend => HTML , CSS , JAVASCRIPT
  - /backend => PYTHON , FastAPI code


    DEPLOYMENT
    
        -/frontend => Netlify
        -/backend => Render

FUTURE POSSIBLE IMPROVEMENTS
   - AI -based job matching
   - Resume upload and parsing
   - Interview scheduling
   - Message system
   - Current status of applications
